financial.py позволяет проанализировать данные Yahoo Finance (https://finance.yahoo.com/quote/msft/financials?p=msft) без использования API.
Необходимо создать виртуальное окружение.
Пример команды для запуска: python3 financial.py 'MSFT' 'Total Revenue'

selects_n_aggs.ipynb обрабатывает сведения и выдает статистику по штрафам.
Запускать в Jupyter notebook
